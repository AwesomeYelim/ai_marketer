# /seo-church — 교회 전문 SEO 캠페인

## 설명
교회/종교기관에 특화된 SEO 캠페인을 실행한다. 도메인 지식을 파이프라인 요청에 주입하고 결과를 교회 맥락에서 평가한다.
Claude Code가 직접 에이전트 역할을 수행한다 (외부 API 키 불필요).

## 사용법
```
/seo-church <교회명> <지역>
```
예시: `/seo-church 사랑의교회 서울 서초구`

## 실행 절차

### Step 1: 도메인 지식 로드
`.claude/skills/seo-church/domain-knowledge.md`를 읽어 교회 SEO 전문 지식을 로드한다.

### Step 2: 에이전트 프롬프트 로드
다음 6개 파일을 모두 읽는다:
- `prompts/knowledge_guide.txt`
- `prompts/planner.txt`
- `prompts/developer.txt`
- `prompts/executor.txt`
- `prompts/trust_manager.txt`
- `prompts/analyst.txt`

### Step 3: 강화된 요청 구성
사용자의 `$ARGUMENTS`에서 교회명과 지역을 추출하고, 도메인 지식을 결합하여 파이프라인 요청을 구성한다:

```
[교회 SEO 캠페인]
교회명: {교회명}
지역: {지역}

핵심 목표:
1. 지역 검색 "근처 교회", "{지역} 교회" 상위 노출
2. 새신자/방문자 유입을 위한 콘텐츠 전략
3. 교회 행사/예배 정보 구조화 (Church JSON-LD)
4. 네이버 블로그/카페 최적화
5. 교회 브랜드 아이덴티티 확립

추가 컨텍스트:
- 예배 시간, 주요 프로그램, 위치 정보를 포함할 것
- 새신자 환영 메시지와 방문 안내를 핵심 CTA로
- 부활절/성탄절 등 교회력 시즌 키워드 포함
- YouTube 설교 영상 SEO 고려

브랜드 아이덴티티:
- 교회 비전/미션을 USP로 활용 (예: "사랑으로 세상을 변화시키는 공동체")
- 교회 성격 키워드 도출 (따뜻한/활기찬/경건한/개방적 등)
- 담임목사/교회 고유 메시지를 브랜드 약속으로 설정
- 환영하는 분위기, 공동체성을 브랜드 보이스에 반영
```

### Step 4: 파이프라인 실행 (DAG 순서)
강화된 요청을 초기 입력으로 하여, 각 에이전트의 시스템 프롬프트를 역할 지침으로 삼아 순차적으로 결과를 생성한다.
도메인 지식(domain-knowledge.md)의 키워드, JSON-LD 스키마, 퍼널, 전략을 각 에이전트 실행 시 참고한다.

**Layer 1 — knowledge_guide:**
- 입력: 강화된 요청
- 출력: 교회 관련 키워드 리서치, 타겟 분석, 전략 수립
- 도메인 지식의 키워드 카테고리(지역 검색, 예배/프로그램, 새신자, 시즌) 참고

**Layer 2 — planner:**
- 입력: knowledge_guide 결과
- 출력: 콘텐츠 아웃라인, 실행 계획

**Layer 3 — developer + executor:**
- **developer**: 메타태그, Church JSON-LD 스키마, 사이트맵
- **executor**: 블로그 글, 소셜미디어 카피, 설교 요약 콘텐츠

**Layer 4 — trust_manager:**
- E-E-A-T 평가 (점수 < 6이면 Layer 3 재실행, 최대 2회)

**Layer 5 — analyst:**
- KPI 예측, 최적화 제안

각 에이전트 결과는 `=== [agent_name] START/END ===` 구분자로 출력한다.

### Step 5: 교회 도메인 특화 평가
일반 파이프라인 평가에 추가로:

- **Church JSON-LD**: `@type: Church` 스키마가 포함되었는가?
- **지역 SEO**: LocalBusiness 마크업, 주소/전화번호/예배시간 구조화
- **새신자 퍼널**: 검색 → 웹사이트 → 방문 → 정착 경로 설계
- **네이버 블로그**: 교회 소식, 설교 요약 콘텐츠 전략
- **YouTube SEO**: 설교 영상 제목/설명/태그 최적화
- **계절 키워드**: 부활절, 성탄절, 추수감사절 대응 여부

### Step 6: 리포트 출력
```

### 캠페인 점수: X/10

### 교회 특화 체크리스트
- [ ] Church JSON-LD 스키마
- [ ] 예배 시간 구조화 데이터
- [ ] 새신자 환영 페이지 콘텐츠
- [ ] 네이버 블로그 전략
- [ ] YouTube 설교 SEO
- [ ] 시즌별 키워드 대응
- [ ] 교회 브랜드 포지셔닝 (비전/미션 기반 USP)
- [ ] 브랜드 보이스 가이드라인 (따뜻하고 환영하는 톤)
- [ ] Organization JSON-LD (교회명, 로고, 소셜 프로필)

### 에이전트별 평가
(표 형식)

### 교회 SEO 개선 제안
1. ...
```
