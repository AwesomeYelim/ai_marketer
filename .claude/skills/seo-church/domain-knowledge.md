# 교회 SEO 도메인 지식

## 핵심 키워드 카테고리

### 지역 검색 키워드
- "{지역} 교회", "{지역} 교회 추천", "근처 교회"
- "{지역} 주일예배", "{지역} 성경공부"
- "교회 찾기", "좋은 교회 추천"

### 예배/프로그램 키워드
- "주일예배", "수요예배", "금요기도회", "새벽기도"
- "주일학교", "청년부", "장년부", "교회학교"
- "성가대", "찬양팀", "소그룹", "셀모임"

### 새신자 키워드
- "교회 처음", "교회 등록", "새신자 환영"
- "기독교 입문", "성경 배우기"
- "교회 분위기", "교회 선택 기준"

### 시즌 키워드
- **부활절** (3-4월): "부활절 예배", "부활절 행사", "부활절 의미"
- **성탄절** (12월): "성탄절 예배", "크리스마스 예배", "성탄 축하 행사"
- **추수감사절** (11월): "추수감사절 예배", "감사절 행사"
- **사순절/고난주간**: "사순절 묵상", "고난주간 특별예배"

## 새신자 유입 퍼널

```
인지 → 관심 → 방문 → 정착

1. 인지: 검색 노출 (지역 키워드, 네이버 블로그)
2. 관심: 웹사이트 방문 (예배 안내, 교회 소개 영상)
3. 방문: 첫 방문 유도 (환영 페이지, 오시는 길, 주차 안내)
4. 정착: 재방문 (소그룹 안내, 새신자반, 주간 소식)
```

## 네이버 블로그/카페 전략

### 블로그 콘텐츠 유형
1. **설교 요약**: 주일 설교 핵심 내용 정리 (매주)
2. **교회 행사 후기**: 사진과 함께 행사 리뷰
3. **성경 묵상 시리즈**: 일일/주간 묵상 가이드
4. **새신자 안내**: 교회 소개, FAQ, 방문 가이드
5. **지역 봉사 활동**: 교회의 지역사회 기여

### 네이버 최적화
- 네이버 지도 등록 (교회 위치, 예배 시간)
- 네이버 블로그 정기 포스팅 (주 2-3회)
- 키워드 태그 활용: #교회 #예배 #{지역}교회

## YouTube 설교 SEO

### 최적화 요소
- **제목**: "[교회명] {날짜} 주일설교 - {설교 제목}" (60자 이내)
- **설명**: 설교 요약 + 타임스탬프 + 교회 정보 + 링크
- **태그**: 교회명, 목사님 이름, 설교 주제, 성경 구절
- **썸네일**: 설교 제목 + 목사님 사진 + 교회 로고
- **자막**: 한국어 자막 필수 (접근성 + SEO)
- **재생목록**: 설교 시리즈별 분류

## 경쟁 환경
- 대형교회: 브랜드 파워로 검색 상위 (차별화 필요)
- 중소형교회: 지역 키워드에서 경쟁 기회
- 핵심 차별화: 교회 고유 프로그램, 목사님 특성, 지역 봉사
