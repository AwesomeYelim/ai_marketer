# /seo-review — 파이프라인 출력 심층 리뷰

## 설명
이전에 실행된 파이프라인 출력을 심층 리뷰한다. 파이프라인을 다시 실행하지 않고 기존 결과만 분석한다.

## 사용법
```
/seo-review <리뷰 대상 또는 관점>
```
예시: `/seo-review 기술 SEO 중심으로 점검해줘`

## 실행 절차

### Step 1: 직전 파이프라인 출력 확인
현재 대화에서 가장 최근의 파이프라인 실행 결과(`=== PIPELINE_RESULT START/END ===` 블록)를 찾는다.
결과가 없으면 사용자에게 먼저 `/seo-run`을 실행하라고 안내한다.

### Step 2: 5개 관점 심층 리뷰

#### 2-1. SEO 기술 컴플라이언스
- 메타태그: title (50-60자), description (150-160자), OG tags 완비
- JSON-LD: 스키마 타입 적절성, 필수 필드 누락 여부
- URL 구조: slug SEO 친화성
- 사이트맵: priority, changefreq 적절성
- 모바일 최적화 언급 여부

#### 2-2. 콘텐츠 품질
- 블로그 글 길이 (2000자 이상 권장)
- H2/H3 구조 적절성
- 키워드 밀도 (1-3% 권장)
- CTA(Call to Action) 포함 여부
- 내부/외부 링크 제안 여부

#### 2-3. E-E-A-T 평가
- Experience (경험): 실제 경험 기반 콘텐츠인가?
- Expertise (전문성): 전문적 깊이가 있는가?
- Authoritativeness (권위): 신뢰할 수 있는 정보인가?
- Trustworthiness (신뢰): 정확하고 투명한가?
- Trust Manager의 평가와 일치하는지 교차 검증

#### 2-4. 한국 검색 시장 특화 (Naver/Kakao)
- 네이버 블로그 최적화 고려 여부
- 네이버 스마트블록 노출 전략
- 카카오 검색 (Daum) 대응
- 한국어 키워드 자연스러움
- 한국 사용자 검색 패턴 반영

#### 2-5. 소셜미디어 플랫폼 적합성
- Instagram: 해시태그 전략, 시각적 요소 언급
- Twitter/X: 글자 수 제한, 바이럴 요소
- LinkedIn: B2B 톤, 전문적 가치 전달
- YouTube: 영상 SEO 키워드, 썸네일 제안

### Step 3: 리뷰 리포트 출력

```
## SEO 심층 리뷰 결과

### 종합 등급: A/B/C/D/F

### 1. 기술 SEO (X/10)
- ...

### 2. 콘텐츠 품질 (X/10)
- ...

### 3. E-E-A-T (X/10)
- ...

### 4. 한국 시장 최적화 (X/10)
- ...

### 5. 소셜미디어 (X/10)
- ...

### 우선 개선 사항 (Top 3)
1. [긴급] ...
2. [중요] ...
3. [권장] ...

### 프롬프트 개선이 필요한 에이전트
- agent_name: 개선 사유
```
