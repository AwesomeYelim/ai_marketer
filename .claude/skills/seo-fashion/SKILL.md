# /seo-fashion — 의류/브랜드 전문 SEO 캠페인

## 설명
의류/패션 브랜드에 특화된 SEO 캠페인을 실행한다. 도메인 지식을 파이프라인 요청에 주입하고 결과를 패션 맥락에서 평가한다.
Claude Code가 직접 에이전트 역할을 수행한다 (외부 API 키 불필요).

## 사용법
```
/seo-fashion <브랜드명> [타겟층] [스타일]
```
예시: `/seo-fashion 모다브랜드 20대여성 미니멀`

## 실행 절차

### Step 1: 도메인 지식 로드
`.claude/skills/seo-fashion/domain-knowledge.md`를 읽어 패션 SEO 전문 지식을 로드한다.

### Step 2: 에이전트 프롬프트 로드
다음 6개 파일을 모두 읽는다:
- `prompts/knowledge_guide.txt`
- `prompts/planner.txt`
- `prompts/developer.txt`
- `prompts/executor.txt`
- `prompts/trust_manager.txt`
- `prompts/analyst.txt`

### Step 3: 강화된 요청 구성
사용자의 `$ARGUMENTS`에서 브랜드명, 타겟층, 스타일을 추출하고, 도메인 지식을 결합:

```
[의류/패션 브랜드 SEO 캠페인]
브랜드명: {브랜드명}
타겟층: {타겟층}
스타일: {스타일}

핵심 목표:
1. 코디/스타일 키워드 상위 노출
2. Product JSON-LD 구조화 데이터
3. 무신사/에이블리 등 패션 플랫폼 최적화
4. Instagram/Pinterest 비주얼 전략
5. 시즌별 룩북 콘텐츠 전략
6. 패션 브랜드 아이덴티티 확립

추가 컨텍스트:
- 타겟 고객층의 검색 패턴과 구매 여정 반영
- 시즌(S/S, F/W) 트렌드 키워드 대응
- 인플루언서 마케팅 연계 SEO 전략
- 가격대별 키워드 전략

브랜드 아이덴티티:
- 브랜드 USP 도출 (디자인 철학, 소재, 가격 포지션 등)
- 브랜드 성격 정의 (미니멀/스트릿/클래식/아방가르드 등)
- 타겟층 공감 포인트를 브랜드 메시지에 반영
- 패션 브랜드 보이스 설정 (감각적/세련된/힙한/편안한 등)
- 브랜드 슬로건/태그라인을 콘텐츠 전반에 일관 적용
```

### Step 4: 파이프라인 실행 (DAG 순서)
강화된 요청을 초기 입력으로 하여, 각 에이전트의 시스템 프롬프트를 역할 지침으로 삼아 순차적으로 결과를 생성한다.
도메인 지식(domain-knowledge.md)의 키워드, JSON-LD 스키마, 플랫폼 전략을 각 에이전트 실행 시 참고한다.

**Layer 1 — knowledge_guide:**
- 입력: 강화된 요청
- 출력: 패션 키워드 리서치, 타겟 분석, 전략 수립
- 도메인 지식의 키워드 카테고리(코디/스타일, 아이템, 쇼핑, 트렌드) 참고

**Layer 2 — planner:**
- 입력: knowledge_guide 결과
- 출력: 콘텐츠 아웃라인, 실행 계획

**Layer 3 — developer + executor:**
- **developer**: 메타태그, Product JSON-LD 스키마, BreadcrumbList, 사이트맵
- **executor**: 룩북 콘텐츠, 블로그 글, 소셜미디어 카피

**Layer 4 — trust_manager:**
- E-E-A-T 평가 (점수 < 6이면 Layer 3 재실행, 최대 2회)

**Layer 5 — analyst:**
- KPI 예측, 최적화 제안

각 에이전트 결과는 `=== [agent_name] START/END ===` 구분자로 출력한다.

### Step 5: 패션 도메인 특화 평가
일반 파이프라인 평가에 추가로:

- **Product JSON-LD**: `@type: Product` 스키마 포함 여부
- **패션 플랫폼**: 무신사/에이블리/지그재그 최적화
- **룩북 콘텐츠**: 시즌별 코디 제안 콘텐츠 전략
- **Instagram 전략**: 스타일링 사진, 릴스, 인플루언서 협업
- **구매 전환**: 검색 → 관심 → 구매 퍼널 설계
- **트렌드 대응**: 현재 시즌 트렌드 키워드 반영

- [ ] Instagram 콘텐츠 전략
