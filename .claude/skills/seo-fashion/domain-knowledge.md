# 의류/패션 브랜드 SEO 도메인 지식

## 핵심 키워드 카테고리

### 코디/스타일 키워드
- "{시즌} 코디", "봄 코디 추천", "가을 데일리룩"
- "{스타일} 코디": 미니멀 코디, 캐주얼 코디, 스트릿 코디
- "출근룩", "데이트룩", "여행룩", "면접룩"
- "키작녀 코디", "키큰남자 코디", "통통녀 코디"

### 아이템 키워드
- **상의**: 니트, 맨투맨, 후드, 블라우스, 셔츠, 크롭탑
- **하의**: 와이드팬츠, 슬랙스, 데님, 미니스커트, 롱스커트
- **아우터**: 패딩, 코트, 가디건, 자켓, 트렌치코트
- **원피스**: 미디원피스, 셔츠원피스, 니트원피스
- **악세서리**: 가방, 신발, 모자, 벨트, 주얼리

### 쇼핑 키워드
- "{브랜드} 추천", "{브랜드} 리뷰", "{브랜드} 사이즈"
- "가성비 옷", "1만원대 옷", "쇼핑몰 추천"
- "무신사 추천", "에이블리 추천", "지그재그 추천"
- "{아이템} 추천", "{아이템} 브랜드 순위"

### 트렌드 키워드
- "{연도} 패션 트렌드", "올해 유행 컬러"
- "올드머니룩", "Y2K 패션", "고프코어"
- "셀럽 패션", "연예인 코디", "인스타 패션"

## JSON-LD 스키마

### Product 스키마
```json
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "상품명",
  "image": ["이미지 URL"],
  "description": "상품 설명",
  "brand": {
    "@type": "Brand",
    "name": "브랜드명"
  },
  "offers": {
    "@type": "Offer",
    "price": "39000",
    "priceCurrency": "KRW",
    "availability": "https://schema.org/InStock",
    "url": "상품 URL",
    "seller": {
      "@type": "Organization",
      "name": "판매처"
    }
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.5",
    "reviewCount": "230"
  },
  "material": "면 100%",
  "color": "블랙",
  "size": ["S", "M", "L", "XL"]
}
```

## 패션 플랫폼 최적화

### 무신사
- 브랜드 스토어 입점 및 최적화
- 무신사 스냅 (코디 사진) 활용
- 랭킹 키워드 분석 (카테고리별)
- 무신사 매거진/스타일 피드 노출

### 에이블리
- 20대 여성 타겟 최적화
- 리뷰 사진 활성화 전략
- 에이블리 추천 알고리즘 대응 (판매량+리뷰)

### 지그재그
- 상품명 키워드 최적화
- Z결제 혜택 활용
- 스타일 태그 활용 (미니멀, 캐주얼 등)

### 29CM
- 프리미엄 브랜딩 콘텐츠
- 에디터 큐레이션 활용
- 브랜드 스토리텔링 중심

## 시즌별 룩북 전략

| 시즌 | 시기 | 핵심 아이템 | 키워드 |
|------|------|-----------|--------|
| S/S | 2-4월 | 가디건, 블라우스, 린넨 | 봄 신상, 봄 코디, 가벼운 아우터 |
| Summer | 5-7월 | 반팔, 린넨, 샌들 | 여름 코디, 시원한 옷, 바캉스룩 |
| F/W | 8-10월 | 니트, 코트, 부츠 | 가을 신상, 가을 코디, 레이어드 |
| Winter | 11-1월 | 패딩, 목도리, 기모 | 겨울 코디, 따뜻한 옷, 크리스마스룩 |

### 룩북 콘텐츠 구성
1. **메인 비주얼**: 시즌 키 아이템 착용 사진
2. **코디 제안**: 3-5개 스타일링 조합
3. **아이템 소개**: 각 상품 상세 + 스타일링 팁
4. **사이즈 가이드**: 모델 사이즈 + 핏 설명
5. **구매 링크**: 직접 구매 유도 CTA

## Instagram/Pinterest 전략

### Instagram
- **피드**: 고퀄리티 룩북 사진, 일관된 피드 톤
- **릴스**: OOTD, 코디 변환, 하울 영상
- **스토리**: 신상 알림, 착용 후기, 폴링 이벤트
- **협업**: 인플루언서 시딩, 공동구매

### 해시태그 전략
- 브랜드: #{브랜드명} #{브랜드명}코디
- 스타일: #데일리룩 #ootd #코디추천
- 아이템: #{아이템명} #{카테고리}추천
- 트렌드: #패션스타그램 #{시즌}신상

### Pinterest
- 코디/룩북 핀 정기 업로드
- 보드 구성: 시즌별, 스타일별, TPO별
- Rich Pin 설정 (상품 가격/재고 자동 연동)
- 한국 사용자 증가 추세 활용

## 인플루언서 콘텐츠 전략

### 협업 유형
1. **시딩**: 무료 제품 제공 → 자연스러운 착용샷
2. **스폰서 콘텐츠**: 유료 광고 → 룩북/리뷰 영상
3. **공동 디자인**: 인플루언서 콜라보 제품
4. **어필리에이트**: 전용 할인코드 + 수익 공유

### 인플루언서 선정 기준
- 팔로워 수보다 참여율(engagement rate) 중심
- 브랜드 타겟층과 팔로워 데모그래픽 일치
- 기존 콘텐츠 스타일과 브랜드 이미지 부합
- 마이크로 인플루언서 (1만-10만) 다수 활용 전략

## 구매 전환 퍼널

```
인지 → 관심 → 고려 → 구매 → 재구매

1. 인지: 검색/인스타 노출 (코디 키워드, 해시태그)
2. 관심: 룩북/코디 콘텐츠 소비 (블로그, 릴스)
3. 고려: 상품 상세 + 리뷰 확인 (사이즈, 소재, 후기)
4. 구매: 할인/프로모션 CTA (첫구매 쿠폰, 무료배송)
5. 재구매: 멤버십/리타겟팅 (신상 알림, 스타일링 추천)
```
