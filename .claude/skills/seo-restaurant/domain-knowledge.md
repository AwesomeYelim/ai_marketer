# 음식점/카페 SEO 도메인 지식

## 핵심 키워드 카테고리

### 지역 맛집 키워드
- "{지역} 맛집", "{지역} {업종} 맛집", "{지역} 맛집 추천"
- "{지역} 점심 맛집", "{지역} 저녁 맛집", "{지역} 데이트 맛집"
- "{역명} 맛집", "{역명} 근처 맛집"
- "근처 맛집", "내 주변 맛집"

### 업종별 키워드
- **한식**: 한정식, 백반, 국밥, 찌개, 고기 맛집
- **카페**: 분위기 좋은 카페, 디저트 카페, 브런치 카페, 작업하기 좋은 카페
- **일식**: 초밥, 라멘, 오마카세, 이자카야
- **양식**: 파스타, 스테이크, 브런치, 피자
- **중식**: 짜장면, 중국집, 딤섬

### 상황별 키워드
- "데이트 맛집", "소개팅 맛집", "가족 외식"
- "회식 장소", "단체 예약", "룸 있는 맛집"
- "혼밥 맛집", "1인 식당", "혼술 맛집"
- "배달 맛집", "포장 맛집"

### 카페 특화 키워드
- "공부하기 좋은 카페", "작업하기 좋은 카페", "노트북 카페"
- "애견 카페", "루프탑 카페", "뷰 좋은 카페"
- "디카페인 커피", "스페셜티 커피", "핸드드립"

## 네이버 Place 최적화

### 필수 등록 정보
1. **기본 정보**: 상호명, 주소, 전화번호, 영업시간
2. **카테고리**: 정확한 업종 분류
3. **메뉴/가격**: 전체 메뉴와 가격 등록
4. **사진**: 외관, 내부, 메뉴 사진 (최소 10장)
5. **편의시설**: 주차, 와이파이, 예약, 단체석 등

### 리뷰 관리 전략
- 영수증 리뷰 유도 (방문 리뷰 가중치 높음)
- 블로거 리뷰 협업 (체험단)
- 부정 리뷰 대응: 24시간 내 정중한 답변
- 사진 리뷰 장려 (시각적 신뢰)

### 네이버 스마트플레이스 기능
- 예약 기능 연동
- 네이버 주문 (배달/포장)
- 쿠폰 발행
- 소식 업데이트 (신메뉴, 이벤트)

## 배달앱 최적화

### 배달의민족
- 업소명에 핵심 키워드 포함 (예: "수제버거 맛집 OOO")
- 메뉴 사진 고품질 등록
- 사장님 댓글 적극 활용
- 울트라콜/오픈리스트 활용

### 쿠팡이츠
- 메뉴 설명 키워드 최적화
- 배달 시간 준수 (평점 영향)
- 쿠팡이츠 전용 프로모션

## Instagram 전략

### 콘텐츠 유형
1. **음식 사진**: 자연광, 위에서 촬영, 색감 보정
2. **릴스**: 조리 과정, ASMR, 먹방
3. **스토리**: 일일 메뉴, 오늘의 추천, 비하인드
4. **카루셀**: 메뉴 소개, 매장 투어

### 해시태그 전략
- 지역 태그: #{지역}맛집 #{역명}맛집
- 업종 태그: #{업종} #맛스타그램 #먹스타그램
- 분위기 태그: #데이트맛집 #분위기좋은곳
- 일반 태그: #맛집추천 #오늘뭐먹지 #foodstagram
